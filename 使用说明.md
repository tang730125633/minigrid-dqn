# MiniGrid DQN 项目 - 使用说明

## 系统要求

- **Windows 10/11** 或 **macOS 10.15+**
- **Python 3.8+**（必须预先安装）
- 4GB+ 内存
- 2GB+ 可用磁盘空间

---

## Windows 用户

### 第一步：安装Python
如果还没有安装Python：
1. 访问 https://www.python.org/downloads/
2. 下载 Python 3.11 或更高版本
3. 安装时勾选 **"Add Python to PATH"**

### 第二步：运行项目
1. 解压文件夹
2. **双击 `安装并启动.bat`**
3. 等待自动安装依赖（首次约3-5分钟）
4. 浏览器自动打开可视化界面

---

## macOS 用户

### 第一步：安装Python（如果还没有）
```bash
brew install python3
```

### 第二步：运行项目
1. 解压文件夹
2. **右键 `安装并启动.command` → 打开**
3. 如果提示"无法打开"，请前往：
   - 系统设置 → 隐私与安全性 → 仍要打开
4. 等待自动安装依赖
5. 浏览器自动打开可视化界面

## 常见问题

### Q1: 安装时提示 pip 错误？
**解决**: 重新运行启动脚本，或手动执行：
```bash
python -m pip install --upgrade pip
```

### Q2: 浏览器没有自动打开？
**解决**: 手动访问 http://localhost:8501

### Q3: 运行很慢？
**原因**: CPU训练深度学习模型较慢
**建议**: 查看已有结果即可，无需重新训练

### Q4: 想修改代码？
**配置**: configs/ 目录下的 .yaml 文件
**源码**: src/ 目录下的 .py 文件
